# ConstructStock

## Sistema de Inventario para Empresas de Construcción

ConstructStock es una aplicación móvil desarrollada para dispositivos Android que permite gestionar el inventario de herramientas y materiales utilizados en empresas de construcción.

La aplicación facilita el registro de productos, el control de entradas y salidas, la actualización automática del stock y el seguimiento del historial de movimientos, ayudando a mejorar la organización del almacén y reducir errores en el control de inventario.

---

# Descripción del proyecto

El proyecto tiene como objetivo desarrollar una aplicación Android que permita administrar el inventario de una empresa constructora mediante una interfaz sencilla e intuitiva.

La aplicación reemplaza el control manual realizado en cuadernos o hojas de cálculo, ofreciendo una solución digital que mejora la organización y el acceso a la información desde dispositivos móviles.

---

# Exposición del problema

Muchas empresas de construcción aún realizan el control del inventario de forma manual, lo que ocasiona problemas como:

- Pérdida de herramientas.
- Desconocimiento del stock disponible.
- Errores en el registro de entradas y salidas.
- Retrasos durante la ejecución de las obras.
- Compras innecesarias por falta de control.

ConstructStock busca solucionar estos inconvenientes mediante un sistema digital que permita administrar el inventario de manera rápida, segura y organizada.

# Objetivo general

Desarrollar una aplicación Android que permita controlar el inventario de herramientas y materiales en empresas de construcción, facilitando el registro de productos, las entradas y salidas de almacén y la consulta del stock disponible.

---

# Objetivos específicos

- Diseñar una interfaz sencilla e intuitiva.
- Implementar un sistema de inventario utilizando SQLite.
- Automatizar el control del stock.
- Reducir errores en la administración del almacén.
- Facilitar el acceso a la información desde dispositivos móviles.
