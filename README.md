# ConstructStock

ConstructStock es una aplicación Android para controlar inventario de herramientas y materiales en empresas de construcción.

## Funcionalidades

- Registro de herramientas y materiales.
- Consulta de inventario general.
- Búsqueda de productos.
- Edición y eliminación de productos.
- Registro de entradas de inventario.
- Registro de salidas de inventario.
- Actualización automática de stock.
- Reporte visual de stock bajo.
- Historial de movimientos con fecha, producto, cantidad y responsable.

## Tecnologías

- Android Studio
- Kotlin
- Base de datos local SQLite
- Git y GitHub

## Cómo ejecutar

1. Abrir Android Studio.
2. Seleccionar **Open**.
3. Elegir la carpeta principal del proyecto `ConstructStock_AGP9`.
4. Esperar la sincronización de Gradle.
5. Crear o seleccionar un emulador Android.
6. Presionar **Run**.
